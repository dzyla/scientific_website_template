Protocols

Below you will find protocols used in our lab. Each protocol has a dedicated page so you can easily reference or print it.

Image Processing

Steps for processing cryo-EM images. Import micrographs into RELION. Perform motion correction. Estimate CTF parameters. Pick particles and generate initial model.

Sample Preparation

This protocol describes the steps for preparing cryo-EM samples. Grow cells to mid-log phase. Harvest and wash with buffer. Apply sample to grids and plunge freeze.
